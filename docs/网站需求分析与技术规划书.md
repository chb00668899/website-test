# 网站需求分析与技术规划书（定稿版）

## 一、项目背景与核心目标

- **项目背景**：构建一个包含前端展示与后端管理的个人/专业博客网站。核心挑战在于使用阿里云入门级低配服务器（2核2G/4G）的同时，需实现流畅的视频播放功能。
- **核心目标**：
    - **高性能博客**：利用 Next.js 构建极速响应的静态/动态混合博客。
    - **视频流媒体能力**：通过 **阿里云 OSS + CDN** 架构，实现视频的低成本、流畅播放，完全规避服务器带宽瓶颈。
    - **AI 辅助开发**：基于 TypeScript 全栈技术栈，利用 Cline 等 AI 工具最大化开发效率。
    - **容器化部署**：使用 Docker 实现一键部署，便于在云服务器上维护。

## 二、核心架构策略（关键决策）

为了在低配服务器上实现高性能视频播放，本项目采用 **“动静分离 + 云端托管”** 的架构策略。

| 模块 | 部署位置 | 技术选型 | 说明 |
| :--- | :--- | :--- | :--- |
| **Web 服务（动静）** | **阿里云轻量服务器** (2核2G/4G) | Docker (Next.js + Node.js) | 负责渲染网页、处理API请求、用户登录。 |
| **数据库** | **Supabase** (云端托管) | PostgreSQL | 托管在 Supabase 云端，不占用服务器内存和CPU，保证数据可靠性。 |
| **视频存储与分发** | **阿里云 OSS + CDN** | OSS Standard + CDN | 视频文件存储在 OSS，通过 CDN 边缘节点分发，**不消耗服务器带宽**。 |

## 三、功能需求详解

### 前台功能（面向访客）

- **首页**：文章与视频混合列表，支持分页或无限滚动。
- **博客模块**：
    - **文章详情**：支持 Markdown 渲染，代码高亮，目录导航。
    - **分类/标签**：按技术栈、生活感悟等维度归档。
- **视频专栏（核心特色）**：
    - **视频列表**：展示视频封面、标题、时长、简介。
    - **视频播放页**：集成 HTML5 播放器，对接阿里云 OSS 的 CDN 加速链接，支持倍速、清晰度切换（若后续开启转码）。
- **关于/联系**：个人简介及留言板。

### 后台功能（面向管理员）

- **统一登录**：基于 Supabase Auth 的安全登录。
- **内容管理**：
    - **文章发布**：Markdown 编辑器，支持图片拖拽上传（图片建议也存 OSS）。
    - **视频录入**：
        - **上传**：前端直传阿里云 OSS（利用 OSS 分片上传能力，减轻服务器压力）。
        - **元数据**：上传成功后，自动获取 OSS 文件 URL，并在数据库中保存标题、描述、封面图等信息。
- **系统设置**：网站标题、SEO 配置、管理员密码修改。

## 四、技术栈定案（AI 辅助优化版）

为了最大化 Cline 的辅助效率，我们锁定以下技术栈：

- **编程语言**：**TypeScript**（类型安全，AI 生成代码准确率高）。
- **前端框架**：**Next.js 14+ (App Router)**（React 生态，SEO 友好，全栈能力强）。
- **样式方案**：**Tailwind CSS**（原子化 CSS，AI 写样式极快）。
- **UI 组件库**：**Shadcn/UI** 或 **Ant Design Mobile**（配合 Cline 快速搭建美观界面）。
- **后端/数据库**：**Supabase**（PostgreSQL + Auth + JS SDK）。
    - *理由*：无需在低配服务器上维护 MySQL 进程，极大节省内存。
- **视频存储**：**阿里云 OSS**。
    - *配置*：开启公共读或通过签名 URL 访问，绑定自定义域名并配置 CDN。
- **部署运维**：**Docker** + **Docker Compose**。

## 五、非功能性需求

- **性能要求**：
    - 网页首屏加载 < 1.5秒。
    - 视频起播时间 < 2秒（依赖 CDN 加速）。
    - 服务器内存占用控制在 60% 以内（依赖 Supabase 卸载数据库压力）。
- **安全性**：
    - 全站 HTTPS。
    - 后台接口鉴权（利用 Supabase RLS 策略）。
    - OSS 防盗链配置（防止视频流量被恶意刷取）。
- **兼容性**：
    - 完美适配移动端（iOS/Android）和桌面端。

## 六、开发实施路线图

1. **阶段一：基础设施与初始化**
    - 购买并配置阿里云服务器（安装 Docker）。
    - 开通 Supabase 项目，获取 API Key。
    - 开通阿里云 OSS，配置 Bucket 和 CDN 域名。
    - 使用 Cline 初始化 Next.js + TypeScript + Tailwind 项目。
2. **阶段二：数据库与后端逻辑**
    - 设计 Supabase 数据表（Posts, Videos, Users）。
    - 编写 Supabase 客户端工具类。
    - 实现登录认证逻辑。
3. **阶段三：前端页面与 OSS 对接**
    - 开发首页、文章页、视频页 UI。
    - **核心攻关**：实现前端直传 OSS 组件（使用阿里云 OSS SDK）。
    - 开发后台管理面板。
4. **阶段四：Docker 部署与联调**
    - 编写 `Dockerfile` 和 `docker-compose.yml`。
    - 在服务器上运行容器，配置 Nginx 反向代理。
    - 进行真机视频播放测试。

## 七、成本预估（月度）

- **阿里云轻量服务器**：约 ¥60 - ¥100/月（固定成本）。
- **Supabase**：免费版（$0/月，额度足够个人博客使用）。
- **阿里云 OSS + CDN**：按量付费。
    - 存储费：极低（¥1-5/月）。
    - 流量费：取决于视频观看量（预估 ¥10-50/月，若流量大需设置上限预警）。
- **域名**：约 ¥60/年。
